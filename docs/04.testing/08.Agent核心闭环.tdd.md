## 用户旅程

作为固定测试用户，我希望创建一个绑定模型版本的会话，以便后续模型发布不会改写当前会话的执行模型。

作为固定测试用户，我希望提交一条带 UUID 幂等键的消息，以便网络重试不会重复执行模型调用。

作为会话编排层，我希望把模型输出校验为诊断和行动结构，以便只保存可执行的行动卡和安全错误。
