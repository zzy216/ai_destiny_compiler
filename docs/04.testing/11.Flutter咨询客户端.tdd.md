## 用户旅程

1. 作为普通用户，我希望在 Flutter 客户端输入 API 地址、账号和密码登录，以便使用自己的咨询数据。
2. 作为已登录用户，我希望看到可用模型，并用默认模型发起一次咨询。
3. 作为已登录用户，我希望发送问题后看到用户消息和法典教练的流式回复。
4. 作为已登录用户，我希望查看历史会话并打开消息，以便回到过去的问题上下文。
5. 作为已登录用户，我希望在设置中看到当前账号和 API 地址，并能退出登录。
