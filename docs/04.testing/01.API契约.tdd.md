> 执行日期：2026-07-16  
> 源计划：[MVP 下一步实施计划](../03.plan/01.MVP下一步实施计划.md)  
> 实现范围：认证、模型、会话 DTO / Controller / Swagger 契约

## 1. 用户旅程

1. 作为 Flutter 或 React 开发者，我希望从同一份 OpenAPI 获取认证、模型和会话接口，以便不自行猜测字段。
2. 作为用户，我希望创建会话时明确选择模型，以便历史会话固定实际模型版本。
3. 作为用户，我希望消息重试携带 UUID 幂等键，以便避免重复消息和重复 Agent Run。
4. 作为用户或管理员，我希望模型查询只返回凭据状态和脱敏提示，以便 API Key 永不回显。
5. 作为 API 使用者，我希望非法字段和格式返回统一错误结构，以便客户端稳定处理错误。
