# 命运编译器 Flutter 客户端

阶段 6 咨询客户端，提供普通用户登录、可用模型选择、咨询对话、历史会话和个人设置。

## 运行

```bash
flutter pub get
flutter run
```

默认 API 地址为：

```text
http://127.0.0.1:3000/api
```

真机调试时，登录页可改为本机局域网地址，例如 `http://192.168.x.x:3000/api`。

## 验证

```bash
flutter test
flutter analyze
```

## 当前范围

- 登录并保存运行期会话。
- 读取当前用户可用模型。
- 创建咨询会话并通过 SSE 接收助手回复。
- 查看历史会话并打开消息。
- 查看当前用户和 API 地址，支持退出登录。
